Duck components are a library of tools to help drawing on the canvas.

Getting started

To get started, we need to use a bit of cotton to wait for a ready canvas

var line = function(){};

var ready = function(){
var canvas = new cotton.duck.objects.Surface('main');

canvas.load(['NullPoint'], function(o){
    NullPoint = o.NullPoint;
    canvas.load(['Line'], line);
});

cotton.ready(ready)

In this example, we wait for a ready event and call the 'line' function.

Live Loading

If your using the live loader, you call the Duck element's you'd like to use prior to usage.

Pass an array and handler to the load() method

var line = function(){};
canvas.load(['NullPoint','Line'], function(){});

Component usage

The API is super simple All components follow the following structure

  • First required argument are NullPoint's
  • Second optional argument is a data object
data = {};
points = [
    new NullPoint(10, 10),
    new NullPoint(10, 100)
];
new Line(points, data);
